Welcome to my blog! This is my first post.

## What's this about?

I'm building a personal blog with Next.js. It's simple, fast, and uses markdown for content.

## Features

- 📝 Write posts in markdown
- 🔍 Search and filter by tags
- 🌙 Dark mode support
- ⚡ Fast and minimal

## Getting started

You can write new posts by going to the `/write` page and using the markdown editor. Posts are stored as `.md` files in the `/posts` folder.

```javascript
function hello(name) {
  console.log(`Hello, ${name}!`);
```
